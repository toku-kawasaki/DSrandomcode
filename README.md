# DSrandomcode
これはDSチートバグ専用のランダムチートコード作成機です。
---
# how to use?
1. [Python](https://www.python.org/)をインストールしてください。
1. `pip install pyperclip`を実行してください。
1. `python dsrandom.py`を実行してください。
1. ビットと範囲を入力するだけでparコードがクリップボードにコピーされるのでそれをチートコードの欄に貼り付けてください。
